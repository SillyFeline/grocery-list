# Grocery List
A simple python program that allows you to create a grocery list and perform CRUD operations on it!
(C - create, R - read, U - update, D - delete)

# Usage
It's very simple! Just clone the repository and run it with python! 
```
git clone https://github.com/SillyFeline/grocery-list.git
cd grocery-list
python main.py
```
###### Or use python3 if needed!
